## Flowsphere

Lots of little dots, doing their thing inside a 3D flow-field ⡐⠑⡡⠓⠔⠕

Built with [Processing 3](https://processing.org/).

### How to

Open the sketch folder in the Processing IDE and press Run.

Drag to rotate. Scroll to zoom.

### Knobs and dials

Mess with `settings.pde`.
